# Diary

Довідник заходів містить дані про дату, час, тривалість, місце проведення, виконавця заходу. Функціональні вимоги: автоматичне нагадування найближчої
справи за поточною датою і часом; видалення вчорашніх справ або перенесення на майбутнє, аналіз «накладок» - перехрестя запланованих справ, перегляд справ на завтра, післязавтра тощо, редагування даних про справи.
